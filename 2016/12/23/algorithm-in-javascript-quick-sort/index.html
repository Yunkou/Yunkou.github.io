快排，理解起来比较“麻烦”点, 见图示

  1. 随便找个数作为基数 图为7
  2. 从两边开始找，左边负责找比基数大的数，找到停止，右边找到比基数小的数停止
  3. 把两个找到的数交换位置，重复2 直到左右相遇
  4. 这时候，基数归位，这时候数列分成两个数列，左边比基数小，右边比基数大。
  5. 分别在两个数列重复1-4

在排序算法中遇到了“二分”这个思想，二分会在以后的学习中遇到。
